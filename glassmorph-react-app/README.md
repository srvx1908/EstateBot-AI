# Glassmorphism React App

This project is a React application that implements a modern 'Glassmorphism' UI design. It features a colorful gradient background, glass effects for various elements, and animations using Framer Motion, while maintaining all existing functionality.

## Features

- **Glassmorphism UI**: The application utilizes a glass-like effect for various components, providing a modern and visually appealing interface.
- **Responsive Design**: The layout is designed to be responsive, ensuring a good user experience on both desktop and mobile devices.
- **Data Visualization**: The app displays data through interactive charts using Chart.js.
- **Chat Interface**: Users can interact with a chatbot that analyzes real estate data based on user queries.
- **CSV Download**: Users can download data in CSV format for further analysis.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **React Bootstrap**: For responsive design and pre-built components.
- **Chart.js**: For rendering charts and visualizations.
- **Framer Motion**: For adding animations to components.
- **Vite**: A build tool that provides a fast development environment.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/glassmorph-react-app.git
   ```

2. Navigate to the project directory:
   ```
   cd glassmorph-react-app
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to view the application.

## Usage

- Enter a query in the input field to analyze real estate data.
- The chatbot will respond with a summary, charts, and a data table.
- You can download the displayed data as a CSV file.
